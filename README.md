# Sistema Bancário Simples

Este é um sistema bancário básico desenvolvido em Python, permitindo operações simples como saque, depósito, investimento e consulta de extrato. O sistema roda em loop contínuo até que o usuário escolha sair.

## Funcionalidades

- **Saque**: Permite sacar um valor respeitando o saldo disponível, limite por saque e limite diário de saques.
- **Depósito**: Permite depositar valores a partir de R$ 1,00.
- **Investimento**: Permite investir valores a partir de R$ 10,00.
- **Extrato**: Exibe todas as movimentações realizadas e o saldo atual.
- **Sair**: Encerra o programa.

## Regras de Negócio

- Saldo inicial: R$ 3.500,00
- Limite por saque: R$ 500,00
- Limite de saques por dia: 3
- Depósito mínimo: R$ 1,00
- Investimento mínimo: R$ 10,00
